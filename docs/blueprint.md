# **App Name**: SponsorLink

## Core Features:

- Tier Selection: Allow sponsors to select a sponsorship tier with predefined benefits and pricing.
- Image Upload: Enable sponsors to upload images for use in social media and printed materials.
- Payment Integration: Integrate with PayPal to allow sponsors to make secure payments.
- Email Notifications: Automatically send an email with all sponsorship details (tier, image, payment confirmation) to randerson@dobmac.com.au after submission.

## Style Guidelines:

- Use the Devonport Tennis Club's primary blue (#00305E) to align with their brand identity.
- Utilize the club's secondary yellow/gold (#FFC20E) as an accent color.
- Light gray (#F5F5F5) or off-white for a clean and professional look, ensuring readability.
- 'Poppins' for headlines, a geometric sans-serif with a precise, contemporary feel; 'PT Sans' for body text, a humanist sans-serif that combines a modern look with a little warmth or personality.
- Single-page layout with clear sections for tier selection, image upload, and payment.
- Use simple, recognizable icons to represent sponsorship tiers and payment methods, maintaining a consistent style with the Devonport Tennis Club's branding.
- Subtle animations to confirm actions like image uploads and payment submissions.
